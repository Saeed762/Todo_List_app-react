# 📝 React To-Do App

A simple and dynamic To-Do application built with React.

## 🚀 Features
- Add new tasks
- Delete tasks
- Toggle task completion (line-through)
- Clean and responsive UI

## 🛠️ Tech Stack
- React (useState, useRef)
- JavaScript (ES6)
- CSS
